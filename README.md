# Luxuria Ecommerce API

Luxuria is a complete **Ecommerce Backend API** built using **Django and Django REST Framework**.  
It provides all essential ecommerce functionalities including product management, cart system, order management, wishlist, reviews, and customer support.

This project is designed to simulate a **real-world ecommerce backend system** with secure authentication and scalable API structure.

---

# Features

## User Authentication
- JWT based authentication
- Secure login & registration system
- Forgot password with OTP verification
- Profile management
- Address management

---

## Product Management
- Product listing API
- Product detail API
- Category & Subcategory system
- Product search and filtering
- Trending products
- Recommended products
- Top rated products

---

## Cart System
- Add to cart
- Update cart quantity
- Remove from cart
- Cart summary API
- Total price calculation

---

## Wishlist System
- Add product to wishlist
- Remove from wishlist
- View user's wishlist

---

## Order Management
- Create order
- Order details API
- My orders API
- Order tracking system
- Payment method support (COD / Online)

---

## Product Reviews
- Add product review
- View product reviews
- Rating system
- Average rating calculation

---

## Deals System
- Discount deals on products
- Active deal detection
- Automatic price calculation after discount

---

## Customer Support System
- Contact support API
- Support ticket threads
- User and admin message system
- Open / closed ticket status

---

## Admin Dashboard
Custom dashboard created for admin management.

Admin can manage:
- Products
- Categories
- Subcategories
- Deals
- Orders
- Customer support tickets
